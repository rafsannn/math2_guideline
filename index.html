3. Sequential Study Roadmap

Follow this step-by-step path to ensure you have the prerequisite knowledge for each new section.

Phase 1: Foundations

Definitions & ODE Formation

Start here. Learn to distinguish ODEs vs PDEs, state order/degree, and eliminate arbitrary constants.
Textbook Ch 1, Pg 1+

Phase 2: Basic Integrations

Variable Separable & Reducible Forms

Master simple integrations where variables can be isolated. Practice substitutions like v = 4x+y+1.
Lecture 1 & 3 Focus

Phase 3: Core Exam Heavyweights

Homogeneous & Linear DEs

The highest yield section. Memorize the Integrating Factor (IF) formula: e∫P dx.
Lecture 5

Phase 4: Advanced Testing

Exact Equations & IF Rules

Check ∂M/∂y = ∂N/∂x. Master the rules to find IF for non-exact equations.
Lecture 6 | Textbook Pg 20

Phase 5: Higher Order (Finals Focus)

Linear DEs with Constant Coefficients

Move to higher derivatives (Dn). Learn Auxiliary Equations to find Complementary Functions (CF).
Lecture 7 | Textbook Pg 60-84
