Practical Syllabus & Core Competencies

Explore the required proficiency areas. Click on the categories below to understand the core methods and required knowledge for the exam.

A

Foundations & Formation

  • Classification: Differentiate between ODE and PDE.
  • Attributes: Identify the Order (highest derivative) and Degree (power of the highest derivative).
  • Formation (Elimination): Practice removing arbitrary constants from families of curves (Circles, Parabolas, Trigonometric functions) to derive their governing DE.
B

First-Order, First-Degree Methods

  • Variable Separable: Direct separation and reducible forms using substitution (e.g., v = ax + by + c).
  • Homogeneous Equations: Utilizing the y = vx substitution.
  • Linear Equations (I.F.): Solving via the Integrating Factor method: I.F. = e∫ P dx.
  • Exact Equations (High Priority): Applying the test for exactness: ∂M/∂y = ∂N/∂x.
C

Higher-Order Linear DE

Constant Coefficients:

Using the Auxiliary Equation (A.E.) to find the Complementary Function (C.F.) and solving for the Particular Integral (P.I.) for RHS functions like eax, cos(ax), or polynomials.

Euler-Cauchy (Homogeneous):

Handling equations of form xn(dny/dxn) by substituting x = ez to transform into constant coefficients.

The "Non-Homogeneous Reducible" Problem: Solve (2x - 2y + 5) dy/dx = x - y + 3
Order/Degree Tasks: Identify values for: (d2y/dx2)3 + (dy/dx)4 + y = 0

Exam Strategy Tips

1

Exactness First

In many exams, a problem that looks difficult is often "Exact." Always check ∂M/∂y = ∂N/∂x first if the equation is in the form Mdx + Ndy = 0.

2

C.F. Cases Memorization

Memorize the three root cases (Real/Distinct, Real/Equal, Complex/Imaginary) for Complementary Functions perfectly. Mixing up real and imaginary root formats is the most common cause of lost marks.

3

Substitution Logic (Euler-Cauchy)

Remember that the operator x2D2 becomes D(D-1) only after the x = ez substitution has been applied. Do not apply it directly to x.
